Buying vs Renting

Analysis based on a 5 year simulation

Baseline

Deposit £30,000
Initial Property Value £200,000
Property Value Yearly Increase 2.5%
Mortgage Term 25 years
Mortgage Interest Rate 5%
Initial Monthly Rent Price £1200
Rent Price Yearly Increase 3%
Savings Interest Rate 5%
If these parameters stayed the same for 5 years, you'd be £56,008 better off if you chose to purchase rather than rent. We'll call this number the delta, where a positive value indicates the buyer is better off and a negative value indicates the renter is better off.
